# System Overview: Talking Canvas

## 1. Architecture
The application is a monolithic web application with a clear separation between the backend API and the frontend client.

- **Backend:** Spring Boot (Java)
- **Frontend:** Angular (TypeScript)
- **Database:** PostgreSQL
- **Containerization:** Docker & Docker Compose

## 2. Backend (Spring Boot)
Located in `src/main/java/com/example/talkingCanvas`.

### Tech Stack
- **Framework:** Spring Boot 3.5.7
- **Language:** Java 21
- **Security:** Spring Security + JWT
- **ORM:** Hibernate / Spring Data JPA
- **Docs:** Swagger / OpenAPI

### Key Packages
- `config`: Security, Swagger, CORS configuration.
- `controller`: REST endpoints for the frontend.
- `model`: JPA Entities mapping to the database.
- `repository`: Data access layer.
- `service`: Business logic.
- `security`: JWT filters and authentication logic.

### Key Controllers
- `AuthController`: Login, Register.
- `PaintingController`: Manage painting listings.
- `OrderController`: Order placement and management.
- `AdminController`: Admin dashboard and management.
- `CartController`: Shopping cart operations.

## 3. Frontend (Angular)
Located in `client`.

### Tech Stack
- **Framework:** Angular 20.3
- **UI:** Angular Material 19.2
- **Language:** TypeScript 5.9

### Structure (`client/src/app`)
- `core`: Singleton services (AuthService, CartService), Guards (AuthGuard, AdminGuard).
- `features`:
    - `auth`: Login, Register.
    - `home`: Landing page.
    - `paintings`: Browse and view paintings.
    - `cart`: Shopping cart.
    - `checkout`: Order placement.
    - `orders`: Order history.
    - `admin`: Admin dashboard (Lazy loaded).
    - `static-pages`: About, Contact, Legal.
- `shared`: Reusable components.

## 4. Database Schema
Tables are auto-generated by Hibernate (`ddl-auto=update`).
Key entities: `User`, `Painting`, `Order`, `OrderItem`, `Cart`, `CartItem`.

## 5. Running the Project
### Docker
```bash
docker-compose up --build
```

### Manual
**Backend:**
```bash
mvn spring-boot:run
```
(Runs on port 8080)

**Frontend:**
```bash
cd client
npm start
```
(Runs on port 4200)
